ساخت یوزر جدید در Django با CMD

برای ساخت یوزر جدید در پروژه Django، دو روش رایج وجود دارد:
روش اول: ساخت ادمین

اگر بخواهی یک کاربر مدیر بسازی، این دستور را در CMD یا ترمینال اجرا کن:

                                                                    bash
python manage.py createsuperuser

بعد از اجرای دستور:

    نام کاربری را وارد کن
    ایمیل را وارد کن
    رمز عبور را وارد کن
    دوباره رمز عبور را تأیید کن

این کاربر می‌تواند وارد پنل مدیریت Django شود.
---------------------------------------------------------------------------------------------------------------------------------------------------------
روش دوم: ساخت کاربر معمولی

اگر بخواهی یک کاربر عادی بسازی، ابتدا وارد shell جنگو شو:

                                                                    bash
python manage.py shell

سپس این کدها را وارد کن:

                                                                    python
from django.contrib.auth.models import User

user = User.objects.create_user(
    username='ali',
    password='12345678',
    email='ali@example.com'
)
user.save()

توضیح خط به خط

    from django.contrib.auth.models import Userمدل User را از جنگو وارد می‌کند.
    create_user(...)یک کاربر جدید می‌سازد.
    user.save()اطلاعات کاربر را در دیتابیس ذخیره می‌کند.

2) English
Creating a new user in Django using CMD

To create a new user in a Django project, there are two common ways:
Method 1: Create a superuser

If you want to create an admin user, run this command in CMD or terminal:

                                                                    bash
python manage.py createsuperuser

After running it:

    enter the username
    enter the email
    enter the password
    confirm the password again

This user can log into the Django admin panel. (in localhost is localhost:{port}/admin
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Method 2: Create a regular user

If you want to create a normal user, first open the Django shell:

                                                                    bash
python manage.py shell

Then type:

                                                                    python
from django.contrib.auth.models import User

user = User.objects.create_user(
    username='ali',
    password='12345678',
    email='ali@example.com'
)
user.save()

Line-by-line explanation

    from django.contrib.auth.models import UserImports Django’s built-in User model.
    create_user(...)Creates a new user account.
    user.save()Saves the user into the database.

) العربية
إنشاء مستخدم جديد في Django باستخدام CMD

لإنشاء مستخدم جديد في مشروع Django، توجد طريقتان شائعتان:
الطريقة الأولى: إنشاء مستخدم مدير

إذا كنت تريد إنشاء مستخدم إداري، شغّل هذا الأمر في CMD أو الطرفية:

                                                                    bash
python manage.py createsuperuser

بعد تنفيذ الأمر:

    أدخل اسم المستخدم
    أدخل البريد الإلكتروني
    أدخل كلمة المرور
    أكد كلمة المرور مرة أخرى

هذا المستخدم يمكنه الدخول إلى لوحة تحكم Django.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
الطريقة الثانية: إنشاء مستخدم عادي

إذا كنت تريد إنشاء مستخدم عادي، افتح أولاً shell الخاص بـ Django:

                                                                    bash
python manage.py shell

ثم اكتب الكود التالي:

                                                                    python
from django.contrib.auth.models import User

user = User.objects.create_user(
    username='ali',
    password='12345678',
    email='ali@example.com'
)
user.save()

شرح السطور

    from django.contrib.auth.models import Userيستورد نموذج المستخدم المدمج في Django.
    create_user(...)ينشئ حساب مستخدم جديد.
    user.save()يحفظ المستخدم في قاعدة البيانات.

ملاحظات مهمة

    من الأفضل استخدام كلمة مرور قوية.
    إذا كان اسم المستخدم موجودًا مسبقًا، سيظهر خطأ.
    تأكد أنك داخل مجلد المشروع قبل تنفيذ الأوامر.
    إذا كان المشروع يستخدم قاعدة بيانات، فيجب أن تكون عمليات الترحيل قد نُفذت مسبقًا.
