فارسی 
ربرای راه انداختن سرور مجازی روی کامپیوترتون شما باید این دستور هارو بزنید ابتیدا ماشین مجازی (venv) رو با این دستور فعال کنید :
venv\Scripts\activate
بعدش هم با این دستور سرور رو ران کنید و لذت ببرید : 
python manage.py runserver
اگر به مشکلی خوردید حتما هم این کارو تو cmd.exe بکنید
-=--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
To run a virtual server on your computer, you need to run these commands: First, activate the virtual machine (venv) with this command: 
venv\Scripts\activate
Then run the server with this command and enjoy: 
python manage.py runserver
If you encounter any problems, be sure to do this in cmd.exe
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
تشغيل خادم افتراضي على جهازك، عليك تنفيذ الأوامر التالية: أولًا، فعّل البيئة الافتراضية (venv) باستخدام الأمر التالي: 
venv\Scripts\activate
ثم شغّل الخادم باستخدام الأمر التالي واستمتع: python manage.py runserver
في حال واجهت أي مشاكل، تأكد من تنفيذ الأمر في موجه الأوامر (cmd.exe).
